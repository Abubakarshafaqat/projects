A lightweight and user-friendly Text Editor built with JavaScript, HTML, and CSS. This editor allows users to create, edit, and save text files with essential formatting options.

Features
✔️ Simple and intuitive interface
✔️ Supports text formatting (bold, italic, underline)
✔️ Save and load text files
✔️ Dark mode for comfortable reading
✔️ Word count and character count
✔️ Keyboard shortcuts for quick editing

Technologies Used
HTML – Structure of the editor
CSS – Styling and themes
JavaScript – Core functionality

Usage
Type in the editor and format your text using toolbar buttons
Click Save to download your html file
Load an existing file using Open
